## üõ† Installation & Usage TradingView

1. Open the file in this repository (e.g., `indicator_name.pine`).
2. Copy the code to your clipboard.
3. Open [TradingView](https://www.tradingview.com/).
4. Click on **Pine Editor** at the bottom of the screen.
5. Paste the code and click **Add to Chart**.
